# プラグイン・コマンドガイド

MARVサーバーで利用できる主要なプラグインとそのコマンドについて解説します。

## 1. プレイヤーユーティリティ
サーバーでの生活を便利にするコマンド群です。

### スコアボード (marvtime)
- `/sbon`: スコアボードを表示
- `/sboff`: スコアボードを非表示

### 経験値管理 (Experience Bottle)
- `/bottle stats`: 現在のXPとレベルを表示
- `/bottle store {量} {個数}`: 指定したXPをボトルに詰める
- `/bottle mend`: 手に持っているツールの修理に必要な分だけボトルを作成
- `/bottle until {レベル}`: 指定レベルまでに必要なXPを表示

### エモート・モーション
- `/sit`: 座る
- `/lay`: 横になる
- `/spin`: くるくる回る
- `/crawl`: 匍匐前進

### 便利機能
- `/ec`: エンダーチェストを開く
- `/nick {名前}`: ニックネームを設定
- `/sethome {名前}` / `/home {名前}`: ホーム設定とワープ

---

## 2. 経済・ショップ
サーバー内での取引と資金管理に関するコマンドです。

### 紙幣・残高管理
- `/balance` / `/money`: 自分の残高を表示
- `/pay {プレイヤー} {金額}`: 他の人に送金
- `/withdraw {金額}`: 指定額の紙幣（アイテム）を発行
- `/deposit`: 手持ちの紙幣を換金（`stack` または `all` で一括換金可能）

### ショップ利用
- `/shop` / `/market`: 公式メインショップを開く
- **QuickShop (チェストショップ)**: 
  - `/qs create`: 手に持っているアイテムでチェストショップを作成
  - `/qs buy`: ショップを購入モードに切り替え
  - `/qs sell`: ショップを販売モードに切り替え
  - `/qs price {金額}`: 価格を設定

---

## 3. 保護・ロック権限
チェストやドアの保護を管理します。

- `/lock`: ブロックやエンティティをロック（自動保護）
- `/unlock`: ロックを解除
- `/bolt trust add {プレイヤー}`: 自分のすべての保護へのアクセスを許可
- `/bolt edit add {プレイヤー}`: 特定の保護ブロックに対してアクセスを許可

---

## 4. 組織 (Organization)
プレイヤー同士でグループ（会社・政党・軍など）を作成できます。

- `/org create {名前}`: 組織を作成
- `/org invite {プレイヤー}`: メンバーを招待
- `/org info {組織名}`: 組織の詳細情報を表示
- `/org setrole {プレイヤー} {役職}`: メンバーにカスタム役職を付与

---

## 5. Towny (町・国家)
大規模な土地保護と国家運営のための主要システムです。

### 町 (Town) コマンド
- `/town create {町名}`: 新しい町を作成
- `/town deposit {金額}`: 町の銀行に入金（維持費の支払いに必要）
- `/town claim`: 現在のチャンクを町の領土として確保
- `/town invite {プレイヤー}`: 住人を招待
- `/town leave`: 町から脱退

### 国家 (Nation) コマンド
- `/nation create {国名}`: 自分の町を中心に国家を建国
- `/nation deposit {金額}`: 国家の銀行に入金
- `/nation invite {町名}`: 他の町を自分の国家に招待
- `/nation ally {国名}`: 他国と同盟を結ぶ

### 区画 (Plot) コマンド
- `/plot forsale {金額}`: 町の土地を住人に売りに出す
- `/plot claim`: 売りに出されている区画を個人用として確保
